# Task To-Do
Task To-Do é um projeto de uma To-Do List criada com JavaScript, onde é possível adicionar, remover e finalizar tarefas.<br>
O foco do projeto é a manipulação do DOM utilizando o JavaScript

# Tecnologias
- HTML
- CSS
- JavaScript
